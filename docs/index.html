Container Database (cdb)

Have your data and use it too!

The container database (cdb) software is a helper utility to build data containers. A data container does not include an operating system, but instead packages just data files along with custom metadata, and provides a means to query it. Data containers can be used in isolation to query data, and then bound as volumes to further utilize the data for an analysis. Data containers can be optimized for a basic search, or even to quickly show workflow or organizational standard metadata. This site will provide a basic overview and examples for getting started.
